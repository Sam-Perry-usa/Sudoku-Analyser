# Sudoku Difficulty + Validity Analyzer (Static)

A small in-browser tool that:
- Validates a Sudoku grid
- Detects whether it has 0 / 1 / multiple solutions (stops at 2)
- Solves using simple techniques (singles, hidden singles, naked pairs)
- Estimates difficulty based on techniques needed

- `1-9` for givens
- `.` or `0` for blanks
